# TGBDual Core: Linking Two Game Boy / Game Boy Color Games

This guide explains how to configure **TGBDual Core** per game in RetroDECK and how to launch linked Game Boy (GB) or Game Boy Color (GBC) games

---

## Set TGBDual as the Emulator in ES-DE

1. In **ES-DE**, highlight the game you want to launch with TGBDual.  
2. Open the **Game Options** menu (press *Select* or open the metadata menu).
3. Go to **Advanced Game Options**.
4. Set **Alternative Emulator** to **TGBDual**.
5. Start the game.

Repeat this per game if you want different games to use TGBDual.

---

## RetroArch Configure TGBDual 

1. Launch the first game via ES-DE.
2. Open the RetroArch Menu → **Subsystems**.
3. Select **Load 2 Player Game Boy Link (GameBoy #1)** → choose the same ROM you launched in `retrodeck/roms/gb` / `retrodeck/roms/gbc` .
4. Go back to **Subsystems** → select **Load 2 Player Game Boy Link (GameBoy #2)** → choose the second ROM.
5. Select **Start 2 Player Game Boy Link** to restart with both games linked.
6. You now have two screens with the games running in parallell.
