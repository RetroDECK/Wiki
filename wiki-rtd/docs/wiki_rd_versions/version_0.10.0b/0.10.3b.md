# RetroDECK v0.10.3b

<img src="../../../wiki_images/logos/rd-logo-box.png" width="600">

**Release date:** 2026-02-03

One more tiny patch of fixes.

---

## RetroDECK Framework

<img src="../../../wiki_icons/retrodeck/icon-framework.svg" width="75">

**Added:**

- **CHEEVOS:** Improved Cheevos login PCSX2 new token handling.

**Changed:**

- **FOLDER ICONS:** Lahrs refined and enhanced 62 folder icons.
- **RETROARCH:** Compressed and optimized the component files by improving path structures and minimizing reliance on symlinks wherever possible.

**Fixed:**

- **RETROARCH:** Restored cores that were missing in the previous stable RetroArch release by reintroducing them from the nightly builds (EasyRPG).
- **STEAM INPUT:** The PS4 layout had incorrect mappings for the Screenshot and Pause hotkeys. The layout version has been updated to `PS4 â€“ DualShock 4 v1.2`.


---

## RetroDECK Configurator

<img src="../../../wiki_icons/retrodeck/icon-configurator.svg" width="75">

**Added:**

- **BIOS CHECKER:** Rewrote the BIOS Checker, reducing the check time from ~1.00-1.30 minutes to about ~2-10 seconds (depending on hardware).
- **BIOS CHECKER:** Added BIOS Checker result logging to `logs/retrodeck_bios_check.log`.

**Fixed:**

- **CLEAN FOLDERS TOOL:**  Fixed Clean Empty Folder Tool behavior to remove folders even when .txt or .icon files are present.
- **GENERAL:** Fixed an issue where the Configurator could exit unexpectedly when clicking OK with no selection. 

---

## Components

<img src="../../../wiki_icons/retrodeck/icon-component.svg" width="75">

**Added:**

- **RETROARCH:** Exposed filters in `retrodeck/shaders/retroarch/filters/`.

**Changed:**

- **RETROARCH:** Relocated the RetroArch shaders directory from `retrodeck/shaders/retroarch/` to `retrodeck/shaders/retroarch/shaders/` to avoid conflicts with the new filters folder.
- **PRIMEHACK:** PrimeHack is once again available as an alternative emulator for Wii and GameCube systems, alongside newly added PrimeHack system entry in 0.10.2b, allowing users to choose their preferred place for the games. Now everyone is happy!


**Fixed:**

- **AZAHAR:** Fixed the switch dual-screen layout hotkey.
- **RETROARCH:** Resolved an error where the shader folders were not set up correctly in certain scenarios.
