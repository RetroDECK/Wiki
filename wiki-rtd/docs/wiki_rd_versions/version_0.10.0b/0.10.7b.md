## RetroDECK Configurator

<img src="../../../wiki_icons/retrodeck/icon-configurator.svg" width="75" alt="RetroDECK Configurator Icon">

**Added:**

- **BIOS CHECKER:** Added support for Triforce BIOS scan.

**Fixed:**

- **BIOS CHECKER:** Some MD5 corrections by community contributor Abdess.
- **RETROARCH BORDERS:** Steam Deck borders were not applied in certain scenarios when enabled via the Configurator.
- **RPCS3:** Firmware installation via the Configurator did not function as expected.

## Components 

<img src="../../../wiki_icons/retrodeck/icon-component.svg" width="75" alt="RetroDECK Component Icon">

**Changes**

- **PPSSPP:** A full configuration reset will occur during the update, due to the addition 400+ lines of new config values.
- **Dolphin:** A full configuration reset will occur during the update, due to the introduction of the new Triforce system.
- **RPCS3:** A full configuration reset will occur during the update, due to major input handling SDL3 changes, evdev removal and a restructure of the config files. <br> **Note:** External controllers will be affected, so adjust settings in **RPCS3 → Pads** via **RetroDECK Configurator → Open RPCS3** for the controllers you are using in your setup.
