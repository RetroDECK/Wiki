## Components

<img src="../../../wiki_icons/retrodeck/icon-component.svg" width="75">

**Added:**

- **RETROARCH:** Exposed filters in `retrodeck/shaders/retroarch/filters/`.

**Changed:**

- **RETROARCH:** Relocated the RetroArch shaders directory from `retrodeck/shaders/retroarch/` to `retrodeck/shaders/retroarch/shaders/` to avoid conflicts with the new filters folder.
- **PRIMEHACK:** PrimeHack is once again available as an alternative emulator for Wii and GameCube systems, alongside newly added PrimeHack system entry in 0.10.2b, allowing users to choose their preferred place for the games. Now everyone is happy!


**Fixed:**

- **AZAHAR:** Fixed the switch dual-screen layout hotkey.
- **RETROARCH:** Resolved an error where the shader folders were not set up correctly in certain scenarios.
