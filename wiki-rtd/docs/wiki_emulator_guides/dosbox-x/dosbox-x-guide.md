# DOSBox-X: General Guide

<img src="../../../wiki_images/logos/dosbox-x-logo.png" width="150">

DOSBox-X is an open-source DOS emulator for running DOS applications, DOS games, DOS-based Windows 3.x and Windows 9x games.

## Where to put the games

DOSBox-X games should be put under the `retrodeck/roms/dos/`, `retrodeck/roms/windows3x/` or `retrodeck/roms/windows9x/` depending on what you want to play.

---

## What file formats are supported?

<img src="../../../wiki_icons/pixelitos/folder-blue-applications.png" width="35">


| File Format | Description |
|-------------|-------------|
| .zip        | Zip Archive |
| .exe        | Windows Executable |
| .bat        | Windows Bat |

---

## Does DOSBox-X require BIOS or Firmware?

<img src="../../../wiki_icons/pixelitos/cpu.png" width="35">

No, but you will need a copy of Windows 98 to play Windows 9x games or Windows 3.1 to play Windows 3.x games.

## Directory structure

<img src="../../../wiki_icons/pixelitos/folder-blue-activities.png" width="35">

| Type   | Directory                                         | Comment       |
|:------:|:--------------------------------------------------|:-------------|
| DOS ROMs   | `retrodeck/roms/dos/`                             |               |
| Windows 3.x ROMs   | `retrodeck/roms/windows3x/`                             |               |
| Windows 9x ROMs   | `retrodeck/roms/windows9x/`                             |               |
| Storage   | `retrodeck/storage/dosbox-x/`                             |    Contains the Hard-drives and other Windows OS data           |
