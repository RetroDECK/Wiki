# RetroDECK: Simple Radial Menu

## How to Access the Simple Radial Menu

- **External Controllers**: Hold `SELECT` and move the `Left Joystick`. Then, press `L3` to activate the selected radial button.
- **Steam Deck**: Use the `Left Touchpad` and press it to activate the selected radial button.

Each radial button corresponds to a specific hotkey or action that helps streamline your gaming experience, such as controlling system functions, game settings, or component-specific features.

---

## Simple Radial Menu

**Column Definitions**

- **Radial Button**: Describes what the hotkey does.
- **Keyboard Command**: Shows what is being sent to the component.
- **System Support**: Lists which systems support the command.
- **Icon**: Displays the icon associated with the action.
- **Comment**: Provides additional notes or explanations for the action.
