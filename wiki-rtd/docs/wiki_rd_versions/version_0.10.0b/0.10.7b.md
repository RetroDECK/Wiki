One tiny patch of fixes and updates.

---

## RetroDECK Framework

<img src="../../../wiki_icons/retrodeck/icon-framework.svg" width="75" alt="RetroDECK Framework Icon">

**Added:**

- **TRIFORCE:** Triforce Arcade System is now supported via Dolphin.
- **EASTER EGGS:** Introduced additional holiday entries.

---

## RetroDECK Configurator

<img src="../../../wiki_icons/retrodeck/icon-configurator.svg" width="75" alt="RetroDECK Configurator Icon">

**Added:**

- **BIOS CHECKER:** Added support for Triforce BIOS scan.

**Fixed:**

- **RETROARCH BORDERS:** Steam Deck borders were not applied in certain scenarios when enabled via the Configurator.
- **RPCS3:** Firmware installation via the Configurator did not function as expected.

---

## Components 

<img src="../../../wiki_icons/retrodeck/icon-component.svg" width="75" alt="RetroDECK Component Icon">

**Changed**

- **PPSSPP:** Have done significant configuration changes, requiring a reset of the config.
- **DOLPHIN:** GameCube controller settings were reset to accommodate additional Triforce input mappings on Port 1.
